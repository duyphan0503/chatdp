# Frontend (Flutter)

This directory is reserved for the Flutter multi-platform app (web/desktop/mobile).

Bootstrap steps:
1. Install Flutter SDK (stable channel).
2. Initialize app here: `flutter create .`
3. Add dependencies according to features (auth, chat, calls).
4. Run: `flutter run -d chrome` (or your target platform).
5. Tests: `flutter test` or `flutter test test/path_to_test.dart`.
